# 8 Puzzle Solver

## Setup 
Clone the repo:
```shell
git clone https://github.com/peterkrauz/8-puzzle-solver.git
```
Enter the new directory:
```shell
cd 8-puzzle-solver/
```
Create a Python 3 virtual environment:
```python
python3 -m venv venv
```
Activate the virtual environment:
```python
source venv/bin/activate
```

## Instructions
```python
python puzzle_solver.py
```
